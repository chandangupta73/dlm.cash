KEY FEATURES:
============
- Django-based investment platform frontend
- Multiple Django apps: accounts, blogs, investments, core_communication, dlm
- Comprehensive testing suite with blockchain and KYC testing
- Rich static file collection including admin, dashboard, and UI components
- Multiple virtual environments for development
- Extensive media handling for KYC documents and user uploads
- Professional admin interface with analytics and reporting
- Responsive templates for various user interactions
- Integration with DRF (Django REST Framework) and Swagger documentation

TECHNOLOGY STACK:

- Django (Python web framework)
- SQLite database
- Bootstrap (CSS framework)
- jQuery (JavaScript library)
- CKEditor (Rich text editor)
- DRF-YASG (Swagger/OpenAPI documentation)
- Various UI plugins and components
- Virtual environment management
